Privacy Policy

This policy explains what Intent Mode Browser processes, where that information is stored, and how it is used in the current version of the extension.

What information the extension processes

  • The browsing goal or intent the user enters.
  • Page information needed for scoring, such as the page title, URL, headings, and visible text on supported pages.
  • Extension settings chosen by the user.
  • Session items created by the user or the extension, such as useful pages, notes, visit history, drift events, and summary statistics.

How that information is used

The information above is used only to power the extension’s focus features, including page scoring, drift detection, session memory, local settings, and optional on-page interventions like paragraph highlighting or feed blurring.

Where data is stored

Based on the current code you shared, the extension stores its state in chrome.storage.local. That includes the current intent, settings, and session data. This site does not claim that the extension sends that data to your own servers or to third-party analytics services.

Data sharing

Intent Mode Browser does not state that it sells personal data. In the current code you shared, there is no declared account system, payment system, external analytics SDK, or remote storage endpoint for user session data.

If you later add cloud sync, analytics, sign-in, or any server-side features, this policy should be updated before release.

User control

Users can clear their active session from the extension popup. Removing the extension also removes access to the extension interface, and Chrome storage can be cleared through the browser or extension removal flow.
