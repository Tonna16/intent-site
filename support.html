Support

Need help using Intent Mode Browser? This page covers the basics for installation, setup, and common issues.

Getting started

  1. Install the extension.
  2. Open the popup.
  3. Enter a goal for your browsing session.
  4. Browse normally and let the extension score pages against that goal.

Common issues

  • If a page does not score, try refreshing the tab and opening the popup again.
  • Chrome internal pages and some restricted pages cannot be analyzed.
  • If a page looks too aggressively filtered, adjust the extension’s settings in the options page.
  • If you want a fresh start, clear the current session from the popup.
