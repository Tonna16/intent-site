Settings

Settings overview

Intent Mode Browser includes a built-in options page where users can control how aggressively the extension highlights, saves, and suppresses distracting content.

Available settings

SettingWhat it doesCurrent default
Hide YouTube ShortsSuppresses common Shorts surfaces and links on YouTube pages.On
Blur recommended feedsBlurs common recommendation rails, sidebars, and feed-like containers until intentionally engaged.On
Highlight relevant paragraphsHighlights paragraphs that match saved intent keywords or phrases.On
Auto-save relevant pagesAutomatically saves highly relevant pages into session memory.On
Show drift bannerShows an on-page reminder when the current page appears to drift away from the user’s goal.On

Scoring thresholds

The options page also lets users adjust the numeric thresholds used to map a page score into labels like relevant, maybe, and distraction.

  • Relevant threshold default: 24
  • Maybe threshold default: 10
  • Distraction threshold default: 0

These values are taken from the current extension settings defaults in the code you shared.

Where users change settings

Users can open the extension’s settings or options page from inside the extension. The extension stores those preferences locally in the browser.
