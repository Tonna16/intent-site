Chrome Extension

Browse with a clear goal, not just another open tab.

Intent Mode Browser helps you start a browsing session with a plain-language goal, score pages against that goal, and reduce distractions while you work. It analyzes the current page in your browser and stores session data locally using Chrome extension storage.

What the extension currently does

Intent sessions

Users can enter a browsing goal in the popup and launch an intent-focused session for research, reading, or other online work.

Local page analysis

The extension classifies the active page as more relevant, a partial match, or a likely distraction based on the user’s current goal.

Session memory

It can save useful pages, session notes, drift events, and lightweight visit history to support focus during a browsing session.

On-page interventions

Depending on settings, the extension can blur recommended feeds, hide YouTube Shorts, highlight relevant paragraphs, auto-save relevant pages, and show drift banners.

How it works

  1. Open the extension popup and enter your goal.
  2. The extension parses your goal into a topic, keywords, and likely distractors.
  3. When you visit pages, the content script evaluates visible page text locally in your browser.
  4. The popup shows a focus score, useful pages, notes, and drift moments for the active session.

Important limitations

  • Some Chrome internal pages and other restricted pages cannot be analyzed.
  • The extension needs access to the current tab’s content on supported sites to score pages.
  • The current version stores data locally; it does not advertise cloud sync or account-based features on this site.
